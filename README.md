Mahashe PATH Editor

Mahashe PATH Editor — это удобный графический редактор переменной окружения PATH в Windows.
Позволяет быстро привести PATH в порядок: добавлять новые пути, удалять лишнее, менять порядок и избегать дублей — без ручного ковыряния в системных окнах.

Возможности
Просмотр и редактирование PATH в удобном списке
Быстрое добавление/удаление путей
Очистка дублей
Безопасное применение изменений (без лишних действий)
Зачем это нужно
Windows-окна “Переменные среды” неудобны для больших PATH: сложно искать, сортировать и контролировать порядок. Этот редактор решает проблему нормальным UI.

Установка и запуск
Скачай релиз из Releases
Запусти MahashePathEditor.exe (или как у тебя называется)

Требования
Windows 10/11
